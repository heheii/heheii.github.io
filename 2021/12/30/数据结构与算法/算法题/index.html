雪花算法会生成重复 ID 吗？如何避免？

https://www.jianshu.com/p/71286e89e0c5

  • 其中位数可以自定义
  • 如果能够保证每个节点都有自己的工作机器 id，就不会有重复
    • 当多个节点对应的 workerId 重复时，可能会有重复的 id 生成
    • 所以需要保证每个节点都有自己的 workerId
  • workerId 和时间戳相同时，当生成的 id 过多，序列号用完的情况下
    • 直接阻塞线程，让时间顺延到下一毫秒，再生成 id
