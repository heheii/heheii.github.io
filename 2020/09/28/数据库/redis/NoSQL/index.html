NoSQL

Not Only SQL，非关系型数据库：

1 文档数据库

这些数据库通常将每个键与称为文档的复杂数据结构配对。文档可以包含键数组对、键值对甚至嵌套文档

2 键值存储

每个单独的项都存储为键值对。键值存储是所有 NoSQL 数据库中最简单的数据库

3 列式存储

这些类型的数据库针对大型数据集上的查询进行了优化，它们将数据列存储在一起，而不是行。

4 图形存储

这些存储关于图形、网络的信息，例如社会关系、路线图、交通链接
