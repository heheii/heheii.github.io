String 为什么不可变

为什么 String 要设计成不可变

安全

  • 若 String 对象允许改变,那么将会导致各种逻辑错误（s1 的修改会影响 s2）
    • 尤其是 String 对象会被当作各种参数使用

效率

  • 缓存 hash（修改时去修改缓存 hash 不可以吗？）
  • 适合作为 map 中的 key

使用简单

  • 因为不可变，所以就不用考虑多线程的情况

循环中的 String

  • String：每一次循环都创建一个 StringBuilder 对象

  • StringBuilder：只在循环外创建一个 StringBuilder 对象
